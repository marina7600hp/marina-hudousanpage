# 解約通知フォーム セットアップ手順

SMSで送れる解約通知フォームの仕組み一式です。

## 全体の流れ

```
① 管理者：kaiyaku-link.html で清掃代パターン＋エアコン洗浄代（台数）を選んでURL作成
        ↓ コピーしてSMSで入居者へ送信
② 入居者：kaiyaku.html（フォーム）に入力して送信
   ・賃貸物件は退去立会い希望日時（9時〜17時・第3候補まで、第1候補必須）を入力
   ・退去時の注意事項への同意（必須）／個人情報の取り扱いへの同意（必須）
   ・「本フォームを契約書に定める書面による解約通知と同等として取り扱う」ことへの
     同意（必須）→ 同意の記録がPDF・受付一覧に残り、PDFにもその旨を明記
        ↓ 送信をもって解約通知が完了（受付番号を表示）
③ 自動処理（Google Apps Script）：
   ・解約通知書PDFを作成 → ドライブの指定フォルダに保存
     （「管理物件」「月極駐車場」サブフォルダに自動振り分け）
   ・marina.oomori1@gmail.com へ通知メール（PDF添付）
   ・入居者がメールを入力していれば受付完了メールを自動返信
   ・フォルダ内のスプレッドシート「解約通知受付一覧」に記録
```

## ファイル構成

| ファイル | 役割 |
|---|---|
| `kaiyaku.html` | 入居者向け解約通知フォーム（管理物件用・月極駐車場用の両対応） |
| `kaiyaku-link.html` | 管理者用。清掃代パターンを選んでSMS用URL・文面を作成 |
| `gas/kaiyaku.gs` | Google Apps Script（メール通知・PDF作成・ドライブ保存） |

## セットアップ（初回のみ・約10分）

### 1. Google Apps Script を作成する

1. **marina.oomori1@gmail.com** のGoogleアカウントでログインした状態で
   <https://script.google.com> を開く
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「解約通知フォーム」などに変更
4. エディタに最初から入っているコードを全部消して、
   このリポジトリの **`gas/kaiyaku.gs` の中身を全部コピーして貼り付け**、保存（💾）

### 2. 動作テスト（権限の承認）

1. エディタ上部の関数選択で **`testSetup`** を選んで「実行」
2. 初回は「承認が必要です」と出るので、アカウントを選択して「許可」
   （「このアプリはGoogleで確認されていません」と出たら
   「詳細」→「解約通知フォーム（安全ではないページ）に移動」→「許可」）
3. 実行が終わると：
   - ドライブの指定フォルダに「管理物件」フォルダとテストPDFができる
   - marina.oomori1@gmail.com にテスト通知メールが届く
   - フォルダ内に「解約通知受付一覧」スプレッドシートができる
   ことを確認してください（テストPDFと行は削除してOK）

### 3. ウェブアプリとしてデプロイする

1. 右上の「デプロイ」→「新しいデプロイ」
2. 歯車アイコン →「ウェブアプリ」を選択
3. 設定：
   - 説明：任意（例：解約通知フォーム）
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**（重要！これでないとフォームから送信できません）
4. 「デプロイ」→ 表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/～/exec`）をコピー

### 4. フォームにURLを設定する

1. `kaiyaku.html` をエディタで開く
2. `<script>` 内の上のほうにある

   ```js
   const GAS_URL = '';
   ```

   に、手順3でコピーしたURLを貼り付ける：

   ```js
   const GAS_URL = 'https://script.google.com/macros/s/～/exec';
   ```

3. コミットしてGitHubにプッシュ（GitHub Pagesに反映されるまで数分待つ）

### 5. 送信テスト

1. ブラウザで `https://（GitHub PagesのURL）/kaiyaku.html?t=k&fee=33000` を開く
2. テスト入力して送信 → 受付完了画面・通知メール・ドライブへのPDF保存を確認

## 日々の使い方

1. `kaiyaku-link.html` を開く
2. 種別（管理物件 / 月極駐車場）を選ぶ
3. 管理物件の場合は**清掃代パターンを選択**（または自由入力）
4. **エアコン洗浄代の台数を選択**（通常 12,100円/台・お掃除機能付き 19,800円/台、いずれも税込）
5. 必要なら物件名・部屋番号を事前入力
6. 「SMS文面をコピー」→ メッセージアプリに貼り付けて入居者へ送信

入居者がフォームを送信すると、通知メールが届き、PDFがドライブに自動保存されます。

## カスタマイズ

- **清掃代パターンの金額変更**：`kaiyaku-link.html` 内の `FEE_PATTERNS` を編集
- **エアコン洗浄代の単価変更**：`kaiyaku-link.html` と `kaiyaku.html` の**両方**にある
  `AC_PRICE_NORMAL`（通常）/ `AC_PRICE_AUTO`（お掃除機能付き）を編集
- **退去時の注意事項の文言変更**：`kaiyaku.html`（フォーム表示）と `gas/kaiyaku.gs` の
  `KANRI_TERMS`（PDF記載）の両方を編集
- **通知先メール・保存先フォルダの変更**：`gas/kaiyaku.gs` 冒頭の `CONFIG` を編集
  → 編集後は GASエディタにも貼り直し、「デプロイ」→「デプロイを管理」→ ✏️ →
  バージョン「新バージョン」で**再デプロイ**（URLは変わりません）
- **フォームの項目変更**：`kaiyaku.html` を編集（既存の書面に合わせた調整もここで）

## URLパラメータ一覧（kaiyaku.html）

| パラメータ | 意味 | 例 |
|---|---|---|
| `t` | 種別（`k`=管理物件 / `p`=月極駐車場） | `t=k` |
| `fee` | 退去時室内清掃代（円・税込、管理物件のみ） | `fee=33000` |
| `ac1` | 通常エアコンの台数（12,100円/台・税込） | `ac1=2` |
| `ac2` | お掃除機能付きエアコンの台数（19,800円/台・税込） | `ac2=1` |
| `b` | 物件名/駐車場名の事前入力 | `b=○○ハイツ` |
| `r` | 部屋番号/区画番号の事前入力 | `r=101` |

例：`kaiyaku.html?t=k&fee=33000&ac1=2&ac2=1&b=○○ハイツ&r=101`

※ 受付一覧スプレッドシートの列構成が変わったため、**旧バージョンで既に
「解約通知受付一覧」が作成済みの場合は、そのファイルの名前を変えておいてください**
（次の受付時に新しい列構成のシートが自動作成されます）。
